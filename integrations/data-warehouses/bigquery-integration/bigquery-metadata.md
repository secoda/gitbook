# Metadata Extracted

## Metadata pulled

Secoda pulls the following metadata from BigQuery:

* Tables
  * Name
  * Description
  * Owners
  * Last Updated Timestamp
  * Schema
  * Database
  * Frequent users
* Columns
  * Name
  * Description
  * Type
  * Foreign Key
  * Primary Key
* Column Profile
  * Min
  * Max
  * Median
  * STD Deviation
  * Value Distribution
  * Statistic Value Count
  * Percent Filled&#x20;
  * Unique
* Creation Query
* Common Queries
* Lineage
  * Column -> Column relationships
  * Table -> Table relationships
  * Table -> Dashboard relationships
  * ETL Job -> Table relationships
* Preview of first 50 rows (Optional)

## Metadata pushed

If enabled, Secoda pushes the following metadata to BigQuery:

Tables

* Description

Columns

* Description
