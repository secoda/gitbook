# MotherDuck Metadata Extracted

Secoda pulls the following metadata from MotherDuck:

* Databases
  * Name
* Schema
  * Name
* Tables and Views
  * Name
* Columns
  * Name
  * Type
* Lineage
  * MotherDuck Table <--> MotherDuck Table
  * MotherDuck View <--> MotherDuck Table
