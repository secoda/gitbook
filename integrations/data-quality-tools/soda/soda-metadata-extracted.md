# Soda Metadata Extracted

### What does Secoda extract from Soda?

* Checks (referred to as "Tests" in Secoda)
  * url
  * column
  * datasets
  * definition
  * description
  * status
  * last check value
  * last check run time
  * last updated
  * name
  * owner
