# Metadata Extracted

### What does Secoda extract from Great Expectations?

* Expectation Suite
  * Name
* Expectation
  * Name
  * Columns
* Validation
  * Name
  * Columns
  * Status
  * Error
  * Created At
  * Result
* Lineage
  * Expectations <-> Tables from other sources
  * Validations <-> Tables from other sources
