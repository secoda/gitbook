# Redshift Metadata

Secoda pulls the following metadata from Redshift:

* Tables
  * Name
  * Description
  * Owners
  * Schema
  * Database
  * Frequent users
* Columns
  * Name
  * Description
  * Type
  * Foreign Key
  * Primary Key
* Column Profile
  * Min
  * Max
  * Median
  * STD Deviation
  * Value Distribution
  * Statistic Value Count
  * Percent Filled&#x20;
  * Unique
* Creation Query
* Common Queries
* Lineage
  * Column -> Column relationships
  * Table -> Table relationships
  * Table -> Dashboard relationships
  * ETL Job -> Table relationships
* Preview of first 50 rows (Optional)
