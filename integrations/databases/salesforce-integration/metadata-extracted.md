# Metadata Extracted

Secoda pulls the following metadata from Salesforce:

* S Objects
  * Title
  * Label
  * Fields
    * Title
    * Description
  * \[Supports Previews and Queries]
* Dashboard
  * Title
  * Description
  * Last updated timestamp
  * URL
* Reports
  * Title
  * Description
  * URL
