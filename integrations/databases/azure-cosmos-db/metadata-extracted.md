# Metadata Extracted

Secoda pulls the following metadata from Azure Cosmos DB:

* Databases (Databases are referred to as Schemas in Secoda)
  * Name
* Containers (Containers are referred to as Tables in Secoda)
  * Name
  * Stored Procedures
* Columns
  * Name
