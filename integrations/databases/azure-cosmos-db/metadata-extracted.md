# Azure Cosmos DB Metadata Extracted

Secoda pulls the following metadata from Azure Cosmos DB:

#### NoSQL Cosmos DB

* Databases
  * Name
* Containers
  * Name
  * Stored Procedures
* Columns
  * Name

#### Tables Cosmos DB

* Tables
  * Name
* Columns
  * Name
