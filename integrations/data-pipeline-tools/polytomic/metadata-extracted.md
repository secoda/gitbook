# Metadata Extracted

### What does Secoda extract from Polytomic?

* SyncsName
  * Description
  * Owners
  * Updated At
  * Executions
    * Name (the ID of the execution)
    * Started At (Timestamp)
    * Ended At (Timestamp)
    * Status
