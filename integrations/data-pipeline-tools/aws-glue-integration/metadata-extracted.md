# Metadata Extracted

### What does Secoda extract from Glue?

Secoda pulls the following metadata from Glue:

* Tables/Views
  * Name
  * Description
  * Last Updated Timestamp
  * Usage (Popularity)
  * Schema
  * Database
* Columns
  * Name
  * Description
  * Type
* Lineage
  * Glue Table <-> Glue Table
  * Glue Query <-> Glue Table
  * Glue Query <-> Tables, Dashboards from other sources
