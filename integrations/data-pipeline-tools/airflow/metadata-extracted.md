# Metadata Extracted

## What does Secoda extract from Airflow?

* DAGs (DAGs are referred to as Jobs in Secoda)
  * Name
  * Description
  * Owners
  * URL
  * Runs
    * Name (ID of the Run)
    * Status
    * Created At (Timestamp)
    * Ended At (Timestamp)
* Tasks
  * Name
  * Description
  * Started At (Timestamp)
  * Ended At (Timestamp)
