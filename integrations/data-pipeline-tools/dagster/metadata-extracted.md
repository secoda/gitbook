# Metadata Extracted

### What does Secoda extract from Dagster?

* Assets
  * Name
  * Description
  * Asset type
* Asset groups
  * Name
* Jobs
  * Name
  * Description
  * URL
  * Tags
  * Runs
    * Name (ID of the run)
    * Status
    * Created date (Timestamp)
    * Finished date (Timestamp)
