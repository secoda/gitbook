# Dagster Metadata Extracted

### What does Secoda extract from Dagster?

* Asset
  * Description
  * Asset type
* Asset groups
  * Name
* Jobs
  * Name
  * Description
  * URL
  * Tags
  * Runs
    * Name (ID of the run)
    * Status
    * Created date (Timestamp)
    * Finished date (Timestamp)
* Step
  * Title
  * Start time
  * Finish time
* Repository
  * Title
* Location
  * Title
* Lineage
  * Repository <-> Asset
  * Asset <-> Job
