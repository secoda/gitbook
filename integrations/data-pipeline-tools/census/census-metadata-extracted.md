# Census Metadata Extracted

### What does Secoda extract from Census?

* Syncs&#x20;
  * Name
  * Description (A string describing the source and destination of the job)
  * Sync Runs
    * Started At (Timestamp)
    * Ended At (Timestamp)
    * Status
    * Error
* Lineage from Source
