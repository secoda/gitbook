# Looker Metadata

Secoda pulls the following metadata from Looker:

* Dashboards
  * Name
  * Description
  * Owners
  * Tags
  * Last Updated Timestamp
  * Folder
* Looks
  * Name
  * Description
  * Owners
  * Tags
  * Last Updated Timestamp
  * Folder
* Explores
  * Name
  * Description
  * Tags
  * Fields (Dimensions & Measures)
* Views
  * Name
  * Description
  * Tags
  * Fields (Dimensions & Measures)
* Lineage
  * Dashboard -> Look relationships
  * Explore -> Dashboard relationships
  * View -> Explore relationships
