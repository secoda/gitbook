# Metadata Extracted

### What does Secoda extract from Cluvio?

* Charts
  * Name
  * Description
  * Queries
* Dashboards
  * Name
  * Description
  * Owner
  * URL
  * Last Updated At
* Reports
  * Title
  * Description
  * URL
* Lineage
  * Cluvio Charts  <-> Cluvio Dashboards&#x20;
