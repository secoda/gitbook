# Preset Metadata Extracted

### What does Secoda extract from Preset?

* Charts
  * Title
  * URL
  * Description
  * Tags
* Dashboards
  * Title
  * Description
  * URL
  * Last Updated
  * Tags
* Datasets
  * Title
  * URL
  * Columns
* Lineage
  * External Database -> Preset Dataset -> Preset Charts -> Preset Dashboard
