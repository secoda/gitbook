# Superset Metadata Extracted

### What does Secoda extract from Superset?

* Charts
  * Title
  * URL
  * Description
* Dashboards
  * Title
  * URL
  * Last Updated
* Datasets
  * Title
  * Description
  * URL
  * Columns
* Lineage
  * External Database -> Superset Dataset -> Superset Charts  -> Superset Dashboard
