# Trino Metadata Extracted

Secoda pulls the following metadata from Trino:

* Catalogs
* Schemas
* Tables
* Views&#x20;
  * Definition
* Columns
  * Type
  * Ordinal Position
* Table -> View lineage
  * Table column -> View column lineage
* Preview of first 50 rows (Optional)
